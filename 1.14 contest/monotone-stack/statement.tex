% Name
单调栈

% Legend
给定一个长度为 $N$ 的整数数列 $a_1, a_2, \ldots, a_N$，输出每个数左边第一个比它小的数，如果不存在则输出 $-1$。

% Input format
第一行包含整数 $N$ ，表示数列长度。

第二行包含 $N$ 个整数 $a_1, a_2, \ldots, a_N$，表示整数数列。

% Output format
共一行，包含 $N$ 个整数，其中第 $i$ 个数表示第 $i$ 个数左边第一个比它小的数，如果不存在则输出 $-1$。

% Note
$1 \leq N \leq 10^5$， $1 \leq a_i \leq 10^9$
